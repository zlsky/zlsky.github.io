基本三角型

  • 正弦 sin(soh) = 对边 / 斜边
  • 余弦 cos(cah) = 邻边 / 斜边
  • 正切 tan(tao) = 对边 / 邻边
  • 1弧度=360/　2×pi　=180/pi
  • 1角度=　2×pi　/360=pi/180

30度=30*pi/180=pi/6=0.523
90度=60*pi/180=pi/2=1.57

1.57弧度=1.57*180/pi=1.57*57.3248 = 90
0.523弧度=0.523*180/pi=0.523*57.3248=30
