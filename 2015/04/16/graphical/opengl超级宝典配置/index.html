freeglut

打开freeglut.sln生成release

  1. 把 x:\x\freeglut-2.8.1\include\GL 文件夹下的 .h文件 复制到 x:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\include\GL 目录下，没有GL文件夹的要自己建。

  2. 把 x:\x\freeglut-2.8.1\lib\x86 文件夹下（以及Debug文件夹）的 .dll 文件复制到 C:\Windows\SysWOW64 目录（x86 复制到 System32目录下）。

  3. 把 x:\x\freeglut-2.8.1\lib\x86 文件夹下（以及Debug文件夹）的 .lib文件复制到 x:\x\Microsoft Visual Studio 10.0\VC\lib。

Glew

  1. 把 x:\x\glew-1.11.0\include\GL 文件夹下的.h文件复制到 x:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\include\GL 目录下。

  2. 把 x:\x\glew-1.11.0\lib\Release\Win32 文件夹下的 .lib文件复制到 x:\x\Microsoft Visual Studio 10.0\VC\lib 目录下。

  3. 把 x:\x\glew-1.11.0\bin\Release\Win32 文件夹下的 .dll文件复制到 C:\Windows\SysWOW64 目录下。

Glut

  1. 把 x:\x\glutdlls37beta 文件夹下的 .h文件 复制到 x:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\include\GL 目录下。

  2. 把 x:\x\glutdlls37beta 文件夹下的 .lib文件 复制到 x:\x\Microsoft Visual Studio 10.0\VC\lib 目录下。

  3. 把 x:\x\glutdlls37beta 文件夹下的 .dll文件 复制到 C:\Windows\SysWOW64 目录下。

构建GLTools库

新建一个gltools的win32项目工程, 应用程序类型为”静态库”,去掉”预编译头”

  1. 把 x:\x\SB5\Src\GLTools\src 目录下的 .h文件（不包括glew.h文件）复制到 x:\x\gltools\gltools。

  2. 把 x:\x\SB5\Src\GLTools\include 目录下的 .cpp文件（不包括GL文件夹）复制到 x:\x\gltools\gltools。

  1. 把 .lib文件 复制到 x:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\lib 文件夹下。

  2. 把 .h文件 复制到 x:\Program Files (x86)\Microsoft Visual Studio 10.0\VC\include 文件夹下（注意这次不是GL文件夹）。

注意不要将gltools里面的glew.h,glew.c复制进去编译了,不然后面会出现重定义的情况.
