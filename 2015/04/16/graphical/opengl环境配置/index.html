vs2013配置opengl开发环境教程

下载opengl函数库

配置步骤

  1. 将lib文件夹下所有文件复制到C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC\lib
  2. 将dll文件复制至c:\windows\SysWOW64
  3. 将GLL文件夹下所有文件复制到C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC\include\GL
  4. 项目->*属性->配置（设为”所有属性“）->配置属性->C/C++->预处理器->预处理器定义->编辑->加上 GLUT_BUILDING_LIB
  5. 项目->*属性->配置（设为”活动（Debug）“）->链接器->系统->子系统(设置为（控制台（/SUBSYSTEM:CONSOLE））)
